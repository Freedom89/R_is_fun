The if statement allows you to check for a condition, and the condition must return TRUE or FALSE. Example of conditions can be found below:

Examples:
one <- 1 ; two <-2

Writing your first if statement

Here is an example of an if-statement. (animal == "cat") is the condition, and the round brackets are required.

In addition, notice the curly brackets which are the parenthesis. 

animal <- "cat"
if(animal == "cat"){
  print("your animal is a cat!")
}

## [1] "your animal is a cat!"

Lists

While Loops

While loops is generally used when you want to achieve a task and is uncertain about the steps you need to take. The sturcture is as follows:

condition <- TRUE
while(condition){ #notice the brackets
  #do some stuff 
  #if the condition is fufilled, change it to FALSE and the while loop stops running. 
  }

As an example: 

i=1 ; condition <- TRUE
while(condition){
  print(i) ; i<- i+1
  if(i == 3){
    condition<-FALSE
  }
}

## [1] 1
## [1] 2

Becareful with while loops as you might encounter infinite loops - the loop will run forever since the condition will never be false!

While Loops (Answer)

Question3.2

Answer:

sum_required <- 500
condition <- TRUE
i <- 1 #start from 1
running_sum <- 0
while(condition){
  if(i %% 3 == 0 ){
    running_sum <- running_sum +i
  }
  if(running_sum >= sum_required){
    condition <- FALSE
  }
  i<- i+1
}
print(i)

## [1] 55

Loops (cont)

There are two additional functionality that are useful - break and next .

The break commands basically stops the loops from running while the next command simply moves on to the next iteration of the loop.

"break" command
for( i in 1:4){
  if(i == 3){break}
  print(i)
}

## [1] 1
## [1] 2

"next" command

for( i in 1:4){
  if(i == 3){next}
  print(i)
}

## [1] 1
## [1] 2
## [1] 4
